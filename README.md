# 🛡️ D2Vault — Destiny 2 Vault Manager

A free CLI tool that scans your Destiny 2 vault, checks every weapon against curated god rolls, ranks your armour by stats, and tells you what to keep, dismantle, or farm for.

---

## 🚀 Quick Start (for your friend)

### 1. Install Python 3

**Windows:** Download from [python.org](https://www.python.org/downloads/) — tick "Add Python to PATH" during install.  
**Mac:** `brew install python3`  
**Linux:** `sudo apt install python3 python3-pip python3-venv`

### 2. Get the tool

```bash
# Download
git clone https://github.com/YOUR_USERNAME/destiny-vault-tool.git
cd destiny-vault-tool

# Or if you got a zip file:
# unzip destiny-vault-tool.zip && cd destiny-vault-tool

# Install
python3 -m venv venv
source venv/bin/activate    # Windows: venv\Scripts\activate
pip install -e .
```

### 3. Get a free Bungie API key

1. Go to https://www.bungie.net/en/Application
2. Sign in with your Bungie account
3. Click **Create New App**
4. App Name: `D2Vault` (or anything)
5. OAuth Client Type: **Confidential**
6. Redirect URL: `https://localhost:8080`
7. Scope: leave defaults
8. Click **Create App**, then copy your **API Key**

### 4. First run

```bash
d2vault --setup
```

It'll ask for your API key and display name — that's it.

### 5. Scan your vault

```bash
d2vault scan                # Terminal report
d2vault scan --html         # Terminal + open the HTML dashboard
d2vault scan --html --quick # Skip manifest download (faster after first run)
```

## 📋 What it checks

### Weapons
- Compares every perk on every weapon against **95 curated god rolls**
- Multiple roll variants per weapon (e.g. PvE God Roll + PvP Roll)
- Scores: 100% = exact match, 75% = main traits correct, 50% = one trait
- Bonuses: masterwork, craftable weapons with deepsight requirements
- Verdicts: **KEEP** ✅ / **DISMANTLE** ❌ / **FARM FOR BETTER** ⚠️

### Armour
- Ranks every armour piece by total stat points
- Keeps top 3 copies per slot with best distributions
- Notes masterwork bonuses (+10 to all stats)
- Categorises as spiky/balanced/mixed

### Farming
- Tells you exactly where each weapon drops
- Raids, Nightfalls, Trials, Seasonal content, World drops
- Deepsight crafting requirements and fastest farm methods

## 🎮 Commands

| Command | What it does |
|---|---|
| `d2vault --setup` | First-run wizard (API key, platform, display name) |
| `d2vault scan` | Scan vault + terminal report |
| `d2vault scan --html` | Also generate interactive HTML dashboard |
| `d2vault scan --quick` | Use cached manifest (skip download) |
| `d2vault analyze` | Same as `scan` |
| `d2vault --help` | Full help |

## 🔧 Files

| File | What |
|---|---|
| `god_rolls/weapons.json` | Curated god roll database — edit to add/update rolls |
| `config.yaml` | Your API key + platform (auto-generated by `--setup`) |
| `output/vault_report_*.html` | Interactive HTML dashboard (generated with `--html`) |
| `~/.destiny_manifest/` | Cached Bungie manifest (~200MB, downloaded once) |
